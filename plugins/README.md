## 🔧 Bukkit 插件 (17个)

| 插件名称 | 简介 |
|---------|------|
| **AuraSkills** | 技能系统，为玩家提供等级、属性和技能成长体系 |
| **CoreProtect** | 数据记录与反破坏工具，支持方块操作查询和回滚 |
| **Essentials** | 服务器基础功能插件，提供经济、传送、家等核心功能 |
| **floodgate** | 基岩版支持插件，允许基岩版玩家无需Java正版账号加入 |
| **FreedomChat** | 聊天自由插件，禁用Minecraft的聊天举报系统 |
| **Geyser-Spigot** | 跨平台支持，让基岩版玩家可以加入Java版服务器 |
| **GuizhanLibPlugin** | 鬼斩中文社区插件前置库，为多个插件提供支持 |
| **Hurricane** | 性能优化插件，提供服务器性能监控和优化功能 |
| **LuckPerms** | 权限管理系统，强大的用户组和权限分配工具 |
| **PlaceholderAPI** | 变量占位符库，为插件间变量调用提供统一接口 |
| **Slimefun** | 大型科技魔法扩展，添加大量新机器、工具和资源 |
| **SlimeGlue** | Slimefun兼容性插件，解决与其他插件的冲突问题 |
| **TpaForm** | 表单式传送请求，提供更友好的传送交互界面 |
| **Vault** | 经济与权限API，作为插件间的通用接口层 |
| **ViaBackwards** | 版本兼容，允许旧版本客户端连接新版本服务器 |
| **ViaRewind** | 老版本支持，支持更早版本的客户端连接 |
| **ViaVersion** | 多版本兼容核心，让不同版本客户端都能加入服务器 |

## 💡 功能分类

### **核心功能**
- LuckPerms (权限管理)
- Vault (API接口)
- Hurricane (性能优化)
- Essentials (基础功能)

### **游戏玩法**
- AuraSkills (技能系统)
- Slimefun (科技扩展)
- HuskHomes (家与传送)

### **跨平台支持**
- Geyser-Spigot + floodgate (基岩版互通)
- Via系列 (多版本兼容)

### **管理工具**
- CoreProtect (数据记录)
- FreedomChat (聊天管理)

### **用户界面**
- CommandPanels (GUI菜单)
- TpaForm (表单传送)
- PlaceholderAPI (变量支持)

### **支持库**
- GuizhanLibPlugin (前置库)
- SlimeGlue (兼容性)

## 🔄 插件协同说明

### **Essentials + HuskHomes 协同**
- Essentials 提供经济系统和基础管理功能
- HuskHomes 专注于家和传送系统，提供更好的用户体验

### **跨平台完整支持**
- Geyser + floodgate + Via系列 提供完整的跨版本、跨平台支持
- Java版和基岩版玩家可以无缝共同游戏

### **技能与科技结合**
- AuraSkills 提供角色成长系统
- Slimefun 提供丰富的科技树内容
- 两者结合创造深度的游戏体验
